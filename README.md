# 🧮 Basic Calculator

A simple and responsive calculator built using **HTML, CSS, and JavaScript**. This project performs basic arithmetic operations such as ➕ addition, ➖ subtraction, ✖️ multiplication, and ➗ division through an interactive and user-friendly interface.

## ✨ Features

* 🎨 Clean and intuitive design
* 📱 Responsive layout for different screen sizes
* ➕➖✖️➗ Supports basic arithmetic operations
* ⚡ Instant calculation results
* 🛡️ Error handling for invalid inputs

## 🛠️ Technologies Used

* 🌐 HTML5
* 🎨 CSS3
* ⚙️ JavaScript (ES6)

## 🚀 How It Works

Users can interact with on-screen buttons to enter numbers and operations. JavaScript processes the input and displays the result instantly on the calculator screen.
